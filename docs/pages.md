\# GitHub Pages Deployment



This repo includes a static demo site under `/docs`.



\## Enable GitHub Pages

1\. Go to the repository on GitHub

2\. Click \*\*Settings\*\*

3\. In the left sidebar, click \*\*Pages\*\*

4\. Under \*\*Build and deployment\*\*

&nbsp;  - \*\*Source:\*\* Deploy from a branch

&nbsp;  - \*\*Branch:\*\* `main`

&nbsp;  - \*\*Folder:\*\* `/docs`

5\. Click \*\*Save\*\*



\## Site URL

Once enabled, the site will be available at:



`https://venkatasivanaga.github.io/trade-exception-tracker/`



\## Pages entry points

\- Home: `/index.html`

\- KPIs: `/kpis.html`

\- Setup: `/setup.html`

\- Docs: `/docs.html`
